# Sealed Book(天书)

Q：这是什么？
A：这是一个用于加密解密文本的小工具。
Q：如何使用？
A：天书奇谈（加密）：将原文输入在上方输入框，输入箴言（密码，可选）即可得到天书（密文）；解读天书（解密）：将天书（密文）输入在下方输入框，输入箴言（可选）即可得到原文。
Q：为何无法得到原文
A：密文不全或已被修改，请您自行确定密文完整程度，解密时需保留“曰：”。
